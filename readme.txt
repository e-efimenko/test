I. ЧТО НЕОБХОДИМО ДЛЯ ЗАПУСКА
	1) Docker desktop
	2) Python 3.x со седующими библиотеками:
		elasticsearch==8.19.3
		pandas==2.3.2
		psycopg2==2.9.12
		SQLAlchemy==2.0.51
		streamlit==1.59.1
		streamlit-js-eval==1.0.0
	3) PostgreSQL (csv файл из задания должен быть загружен в базу данных)

II. ЗАПУСК
	1) выполнить в командной строке docker run -p 9200:9200 -e "discovery.type=single-node" docker.elastic.co/elasticsearch/elasticsearch:8.12.0
	2) заполнить файл password.json:
		login и password в объекте elastic это параметры для аутентификации на сервере Elasticsearch. Если они неизвестны, то:
			а) login присвоить значение по умолчанию - elastic
			б) открыть командную строку (не закрывая ту, в которой уже запущен docker) и выполнить docker ps. В консоль будет выведена информация о запущенных докер контейнерах, нужно найти CONTAINER ID запущенного в первом пунтке контейнера. Если запущен только один контейнер, то будет только один CONTAINER ID
			б) выполнить команду docker exec -it id_контейнера_из_предыдущего_пункта bin/elasticsearch-reset-password -u elastic и подтвердить действие ответом "y", значение из строки New value записать в password
		в объекте postgresql
			database это имя базы данных, где хранятся документы
			table это имя таблицы в этой базе
			login и password это логин и пароль от PostgreSQL
		файл password.json должен находится в той же папке, что и test.py
	3) выполнить в командной строке python -m streamlit run путь_к_файлу_test.py, после этого в браузере откроется интерфейс для работы со скриптом. Если этого не произошло, открыть в браузере http://localhost:8501

III. КАК ПОЛЬЗОВАТЬСЯ
	1) для поиска нужно ввести запрос в текстовом поле и нажать кнопку Поиск
	2) для удаления записи из базы и из эластика нужно ввести в текстовом поле id удаляемой записи и нажать кнопку Удалить из базы
